Pizza - SPA учебное приложение на React.

Приложение предоставляет функционал пиццерии.
Spa включает в себя:

    Пагинация
    Skeletons при загрузке страницы 
    Сортировка и фильтрация товаров по разным параметрам.

Стек фронтенда:

    HTML
    SCSS
    React
    Redux Toolkit

Стек бекенда:

    NodeJs
    PostgreSQL
    REST API
